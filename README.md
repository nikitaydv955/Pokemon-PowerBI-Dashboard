# 📊 Pokémon Data Analysis Dashboard

## 📌 Overview
This project presents an interactive **Power BI Dashboard** built using the Pokémon dataset. The dashboard analyzes Pokémon statistics, generations, legendary status, and type-based performance to uncover meaningful insights through data visualization.

---

## 🎯 Objectives
- Analyze Pokémon attributes and performance metrics.
- Compare Attack and Defense statistics.
- Identify the distribution of Legendary Pokémon.
- Explore Pokémon counts across different generations.
- Evaluate the performance of various Pokémon types.

---

## 🛠️ Tools Used
- Power BI
- DAX
- Microsoft Excel / CSV
- Data Cleaning & Transformation
- Data Visualization

## 📈 Dashboard Visualizations

### 🔹 Attack vs Defense
A scatter plot visualizing the relationship between Attack and Defense stats, helping identify strong offensive and defensive Pokémon.

### 🔹 Legendary Pokémon Distribution
A donut chart displaying the percentage of Legendary and Non-Legendary Pokémon.

### 🔹 Pokémon by Generation
A bar chart showing the number of Pokémon introduced in each generation.

### 🔹 Pokémon by Type
A distribution chart showing the count of Pokémon across different primary types.

### 🔹 Average Total Stats by Type
A comparison of average total stats for each Pokémon type to identify the strongest-performing types.

---

## 🔍 Key Insights

- Water-type Pokémon have the highest count in the dataset.
- Only about 10% of Pokémon are Legendary.
- Flying-type Pokémon have the highest average total stats.
- Generation 5 contains the largest number of Pokémon.
- Attack and Defense show a positive correlation.
